# AI智能识别功能说明

## 功能概述

在发布页面上传图片后，点击「AI识别」按钮，系统会自动识别图片中的物品并生成详细描述，自动填入描述框。

## 使用方法

1. 在发布页面上传物品图片
2. 点击图片区域右上角的「🤖 AI识别」按钮
3. 等待AI识别完成（通常需要2-5秒）
4. AI生成的描述会自动填入「详细描述」输入框
5. 可以根据需要修改描述内容
6. 继续完成其他信息填写并发布

## 功能特点

- **智能识别**：自动识别物品类型、颜色、材质、品牌等特征
- **快速便捷**：一键生成描述，节省填写时间
- **准确详细**：AI生成的描述包含物品的主要特征和状态
- **可编辑**：生成的描述可以根据实际情况进行修改

## 识别内容

AI会识别并描述：
- 物品名称和类型
- 颜色和外观特征
- 材质和品牌（如果可识别）
- 物品状态（新旧程度、是否有损坏等）

## 注意事项

1. 需要先上传图片才能使用AI识别功能
2. 建议上传清晰、光线充足的图片以获得更好的识别效果
3. AI识别需要网络连接，请确保网络畅通
4. 识别结果仅供参考，请根据实际情况进行调整
5. 如果识别失败，可以手动填写描述信息

## 技术实现

- 前端：uni-app
- 后端：Node.js + Express
- AI服务：OpenAI Vision API (gpt-4o-mini)
- 接口：POST /api/ai/recognize

## 隐私说明

- 图片仅用于AI识别，不会被永久存储在AI服务器
- 识别过程通过加密连接传输
- 符合数据安全和隐私保护要求
